    Starter kit for single page app

    Demo

    http://yannickbochatay.github.io/JSYG-starterkit/

    Documentation generated with esdoc

    http://yannickbochatay.github.io/JSYG-starterkit/doc

    Features

    • ES6 and JSX with Babel

    • Less and Sass support

    • Code quality with ESLint

    • smart auto-build and reload with WebPack

    • easy imports from bower components and npm packages

    • Testing environment with Karma and Jasmine

    • Esdoc for generating documentation

    • React components

    • React-router for dynamic routing

    • React-intl for internationalization

    • React-redux for state container

    • Grommet for ui components

    Prerequisites

    • NodeJS installed

    Installation

    • download and extract zip

    • move to directory JSYG-starterkit-master

    • enter command

      npm install

      It can takes several minutes

    Get started

    npm start

    Build app (in memory) and launch the dev-server with hot reload. You have to open your browser and go to http://localhost:8080/.

    npm test

    Run tests with Karma and Jasmine. All files with .test.js or .test.jsx extension inside src/ directory will be executed, with hot reload.

    npm run build-dev

    Build app (in builds/ folder) for development

    npm run build-prod

    Build app (in builds/ folder) for production (clean, optimize, minify)

    npm run doc

    Generate documentation (in doc/ folder)

    Warning

    If your app is not in the web-root folder (ex : http://myDomain.fr/myApp/ instead of http://myDomain.fr/ ) you'll have to change the publicPath property in webpack.config.js (ex : publicPath : "/myApp/builds/" instead of publicPath : "/builds/").

    Install new dependencies

    with npm
    npm install --save jquery
    or bower if you prefer
    bower install --save jquery
    in your code
    import $ from "jquery"

$("<div>").appendTo("body");

    That's it.
